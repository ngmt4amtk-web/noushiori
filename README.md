# 脳栞 (noushiori)

md/txt を読み、重要な文を前後文脈ごと「栞」として捕獲し、抜き出して、思い出すための iPhone 向け読書ツール（依存ゼロの PWA）。

公開URL: https://ngmt4amtk-web.github.io/noushiori/

## 使い方（iPhone）
1. Safari で上のURLを開く
2. 共有 → 「ホーム画面に追加」でアプリ化（全画面・オフライン動作）
3. 「＋ 取り込む」で md/txt を選ぶ or 本文を貼り付け

## 読み方（3モード）
- **集中**: 全文を表示したまま現在の文を中央へ誘導。自動ペーシングは自分でタップ起動・即停止。再読・先読み・自分のペースを殺さない（＝理解を壊さない）。
- **フラッシュ**: 中央に文節チャンクを逐次表示（RSVP）。顔や目を動かさず流す拾い読み向け。光がつらい時は集中へ。
- **自由**: ただスクロール。図表・図解の多い本向け。

## 捕獲 → 抽出 → 想起
- 本文をタップで再生/停止、❏栞ボタンで今いる文を前後ごと捕獲、文を長押しで捕獲＋メモ。
- 栞は md / txt / Anki(TSV) で抽出、共有シートで Obsidian 等へ。
- 見出しごとの想起プロンプト＋Leitner 方式の復習で定着。

## 設計の根拠
RSVP 単独は理解（回帰・周辺視・自己ペース）を削るため、知識抽出の土台にはしない。
捕獲・抽出・想起を背骨に据え、表示様式は素材ごとに選べる構成にした。
詳細レビュー: `~/Shortcuts/Claude Code成果物/2026-05-30_高速読書抽出アプリ_壁打ちレビュー.md`

## 技術
vanilla JS / CSS / HTML、IndexedDB（端末内保存、失敗時はメモリ動作）、Service Worker でオフライン、Wake Lock、safe-area 対応。ビルド不要。
